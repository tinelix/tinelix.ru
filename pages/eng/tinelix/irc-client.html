Description

August 2021. So, like Veselcraft, who worked on a Escargot (MSN/WLM) client, I decided to create something similar, but now IRC client on Python. I note that the Qt 5 framework, other packages are used here: PyQt5 - Qt 5 implementation on Python, cryptography for password encryption) и configparser (for file settings like *.ini format) and standard Python 3 modules. Connects to IRC server via socket module. This client supported five encodings - UTF-8, Windows-1251 (for gramsk.ru is actual), DOS 866, KOI8-R and KOI8-U.

Source code repository (GNU GPL 3.0)
Tinelix IRC Client portable versions by Dmitry Ivchenko (aka. HSD)

Please note: to run the flight.py script, download the Python interpreter (3.7+). It is desirable, at the same time, we specify the Add Python [your version] to PATH parameter, and download the missing packages via cmd or the terminal via pip3. For example, pip3 install [package]. You can check it at the very beginning of the flight.py file.

Warning: You need to run the flight.py script, because the rest of the scripts do not work, but they are needed to compose forms and GUI dialogs.
