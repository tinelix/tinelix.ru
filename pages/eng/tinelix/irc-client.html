Tinelix IRC Client

I'll tell you a few words about what it is.

IRC stands for Internet Relay Chat. This is a simple and old protocol for communicating with people in real time. It only uses text data and a slightly different set of text formatting than Markdown and provides socket connectivity to any IRC client. To do this, you need to know only basic commands like PONG, JOIN, MODE, PRIVMSG and QUIT.

And since I started getting acquainted with the http://narodweb.ru site in the early summer of 2021, in August in my head there was such an idea about creating my own IRC client in the Python 3.x programming language and based on PyQt5, since at the moment my colleague Vladimir Barinov (Veselcraft) wrote a client for Escargot servers in JavaScript for the node.js interpreter.

Что нового появилось в Tinelix IRC Client версии 0.1.2?

1. In addition to support for Android 2.x, which appeared with version 0.1.0, the appearance of the application has been improved. This is for Android versions below 3.0.

To some extent, this design vaguely resembles the Holo interface that was in Android 4.x.
2. Added ability to create custom QUIT messages.
3. Fixed bug in Profile Adapter (custom list adapter for ListView) in connection manager activity
4. Added notifications about nickname mentions.
